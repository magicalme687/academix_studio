How to Use — Exam Seating Manager

Follow these 7 steps to generate seating charts, timetables & attendance sheets.

⚙️
Step 1 — Base Configuration

Enter Institute Name (shown as heading on all printed pages).
The Address / Contact text box below it adds address/contact lines under the name in every header (default: IPS Academy address).
Upload optional Institute Logo, select Department & MID SEM number.
Choose Subject Source:
• Upload Excel — Row 1 = headers (I Yr, II Yr…), data starts row 2.
• Manual Entry — Comma-separated codes per year.
Upload the Student List Excel (enrollment + names).
(Tip: Click Download Sample Excel next to the upload buttons to get the template formats.)

📅
Step 2 — Master Timetable Builder

Click Add Another Date, then for each date → Add a Shift:
• Set Start Time (End Time auto-fills).
• Check the years in this shift & pick their subject code.
• A subject used in one shift is disabled in all others automatically.

🏫
Step 3 — Room Layout Configuration

Enter Number of Rooms → click Configure Rooms.
For each room: edit Room Name (auto-increments N101→N102), set Rows & Columns, choose Year Pattern and Entry Door.
Use Batch Configuration to apply the same settings to multiple rooms at once.

🚀
Step 4 — Generate

Click Generate Timetable, Seating & Attendance.
Students are distributed so no two of the same year sit in adjacent columns. Four tabs are generated: Master Timetable, Seating Charts, Master Attendance, and Room-Wise Attendance.

✏️
Step 5 — Edit Seating Seating Charts tab only

Click ✏️ Edit Seating to unlock the layout. A Swap Buffer tray slides up from the bottom.
• Same room: Drag a seat and drop it directly onto another seat in the same room — they swap instantly.
• Different room: Drag a student to the Swap Buffer, switch to the target room tab, then drag the card from the buffer onto the target seat.
• Click ✕ on a buffer card to return just that student to their original empty seat.
• Click ↺ Cancel Changes to revert all edits and restore the full layout to its pre-edit state.
• When the buffer is empty, click ✔ Confirm Seating to lock the layout and refresh all attendance sheets.

🚪
Step 6 — Add Extra Room Seating Charts tab only

Click + Add Room to append a new room post-generation.
Set its name, rows, columns, and seating pattern.
Student data is optional — leave the file upload empty to create a blank room, then populate it by dragging students in from the Swap Buffer via Edit Seating mode.

📤
Step 7 — Export

Switch to the tab you want to export, then:
• Export Excel — Downloads the active tab as a named .xlsx file.
• Print / Save as PDF — Opens print dialog with a descriptive filename. Room charts fit one-per-page.
• Footer Settings: Toggle 'Enable Printable Footer' in the Config section to add signature blocks exclusively to the sheets you choose (Timetables, Seating, or Attendance).
